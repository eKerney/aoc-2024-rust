#### Installing Rust on WSL2 
```bash

# installing rust 

https://www.rust-lang.org/tools/install 

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# default install 
# stable-x86_64-unknown-linux-gnu installed - rustc 1.83.0 (90b35a623 2024-11-26)

```
